# 💰 User Currency Management System

A console-based C program that reads multi-currency user records from a text file, lets you look up individual balances, and exports all data to a CSV file. Built as a university assignment demonstrating file I/O, structs, modular programming, and input validation in C.

---

## What the Program Does

1. **Loads** user records from `coins.txt` — each record contains a name, an amount (1–95), and a currency (US$, AU$, or Euro)
2. **Aggregates** amounts per currency per user — if a user appears more than once, their balances are accumulated
3. **Presents a menu** with three options:
   - View a specific user's currency balances
   - Export all user data to `change.csv`
   - Exit
4. **Validates** all input — amounts outside the 1–95 range and unrecognised currencies are rejected during loading

## Input Format — `coins.txt`

The first line is the total number of records, followed by one record per line:

```
6
Alice 45 US$
Bob 30 AU$
Alice 20 Euro
Carol 95 US$
Bob 50 Euro
Carol 10 AU$
```

- **Name**: single word, no spaces
- **Amount**: integer between 1 and 95 (inclusive)
- **Currency**: must be exactly `US$`, `AU$`, or `Euro`

Records with invalid amounts or unrecognised currencies are skipped.

## How to Compile & Run

### Linux / macOS

```bash
gcc main.c functions.c -o currency_manager
./currency_manager
```

### Windows (MinGW)

```bash
gcc main.c functions.c -o currency_manager.exe
currency_manager.exe
```

Make sure `coins.txt` is in the **same directory** as the executable before running.

## Functions Overview

| Function | File | Description |
|---|---|---|
| `loadUserData()` | `functions.c` | Opens `coins.txt` and reads all records |
| `processLine()` | `functions.c` | Parses and validates a single line |
| `addNewUser()` | `functions.c` | Adds a new user entry to the array |
| `updateUserAmount()` | `functions.c` | Accumulates balance for an existing user |
| `displayUserDetails()` | `functions.c` | Prints a user's balances to the console |
| `saveChangeToCSV()` | `functions.c` | Writes all user data to `change.csv` |
